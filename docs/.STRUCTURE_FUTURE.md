### 1. `backend/config.py` — Centralizar configuración

```python
"""
config.py -- Configuración centralizada y validada de Quimera.
Reemplaza los env vars sueltos en main.py con un objeto validado.
"""

from dataclasses import dataclass
from typing import Optional
import os
from dotenv import load_dotenv

load_dotenv()

@dataclass
class Config:
    """Configuración validada de Quimera."""
    
    # Entorno
    ENVIRONMENT: str = os.getenv("ENVIRONMENT", "development")
    
    # GCP
    GCP_PROJECT_ID: str = os.getenv("GCP_PROJECT_ID", "quimera-ai-prod")
    GCS_BUCKET: str = os.getenv("GCS_BUCKET", "quimera-ai-prod-outputs")
    GOOGLE_API_KEY: Optional[str] = os.getenv("GOOGLE_API_KEY")
    
    # API
    ALLOWED_ORIGINS: list[str] = None
    MAX_ITERATIONS: int = 3
    
    # Modelos
    GEMINI_MODEL_PRO: str = "gemini-2.5-pro"
    GEMINI_MODEL_FLASH: str = "gemini-2.5-flash"
    
    # Validation
    APPROVAL_THRESHOLD: float = 85.0
    
    def __post_init__(self):
        if self.ALLOWED_ORIGINS is None:
            origins_str = os.getenv("ALLOWED_ORIGINS", "*")
            self.ALLOWED_ORIGINS = origins_str.split(",")
    
    def validate(self):
        """Valida que la config sea completa."""
        if self.ENVIRONMENT == "production":
            assert self.GOOGLE_API_KEY, "GOOGLE_API_KEY requerida en prod"
        return True

# Instancia global
config = Config()
config.validate()
```

**Uso en main.py:**

Facilita agregar nuevos scorers sin duplicar código.

from abc import ABC, abstractmethod
from models import DesignContext

class BaseScorerr(ABC):
    """Interfaz para todos los scorers de evaluación estética."""
    
    name: str
    description: str
    
    @abstractmethod
    async def score(self, context: DesignContext) -> tuple[float, str]:
        """
        Puntúa un aspecto del diseño.
        
        Returns:
            (score: float 0-100, explanation: str)
        """
        pass

# Uso:

    name = "wcag_contrast"
    description = "Contraste WCAG 2.1"
    
    async def score(self, context: DesignContext) -> tuple[float, str]:
        # Implementación

## 🎓 Principios de Diseño Aplicados

### 1. **Single Responsibility Principle (SRP)**
Cada archivo tiene UNA responsabilidad:
- `config.py` → Configuración
- `routers/pipeline.py` → Endpoints pipeline
- `pipeline/step1_analyze.py` → Lógica Paso 1

### 2. **Don't Repeat Yourself (DRY)**
- `base.py` → No repetir interfaz scorer
- `config.py` → No esparcir env vars
- `routers/` → No mezclar lógica con servidor

### 3. **Dependency Injection**
```python
# Pasable, testeable
async def analyze_and_design(
    context: DesignContext,
    client: GeminiClient = None  # Default, pero inyectable
) -> DesignContext:
    client = client or GeminiClient()
    # ...
```

### 4. **Clear Boundaries**
```
API Layer (main.py, routers/) 
    ↓
Pipeline Logic (pipeline/, scorers/)
    ↓
Services (gemini, firestore, storage)
    ↓
External APIs (Gemini, GCP)
```
